# 배경 이미지

이 폴더에 배경 이미지를 추가하면 영상 배경으로 사용됩니다.

| 파일명 | 카테고리 |
|--------|----------|
| `morning.jpg` | 아침 커피 타임 |
| `nature.jpg` | 명상 음악 |
| `night.jpg` | 수면 음악 |
| `study.jpg` | 공부 집중 음악 |
| `rain.jpg` | 비오는 날 음악 |

## 권장 사양
- 해상도: **1920 × 1080** (Full HD)
- 포맷: JPG 또는 PNG

이미지가 없으면 카테고리별 그라디언트 배경이 자동 생성됩니다.
